FlashFlip is a tool for creating flashcards through AI, where the user can provide context to the AI, and then ask it to generate flashcards based on that context. The user can also create their own flashcard sets.

## How It Works

The user can create a flashcard set by providing a name for the set, and then adding flashcards to the set. Each flashcard consists of a question and an answer. The user can also provide context to the AI, which can be used to generate flashcards automatically. For example, I used it by providing my notes for a course, and then asking the AI to generate flashcards based on those notes. The AI was able to generate flashcards that were relevant to the course material, and that helped me study more effectively.

## Technical Implementation

FlashFlip is built using Vue.js, and uses a Supabase backend for data storage and API endpoints. The AI flashcard generation is powered by OpenAI's GPT-4o mini, which allows it to parse the context provided by the user and generate relevant flashcards.
